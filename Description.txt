В программе ввод матриц происходит из файлов(и числа должны быть разделены двумя пробелами, извините), матрица-результат выводится на экран. 
Для работы с диалоговыми окнами, а также с файлами прописаны интерфесы и реализующие их классы(папка FileInteraction). 
Папка DataTransformation предоставляет класс для преобразования матрицы из вида, получаемого из представления, в вид, необходимый модели.
Папка Command содержит класс, объекты которого создаются в представлении модели для привязки к кнопкам представления.
Модель представляет собой класс, способный на перемножение матриц в многопоточном режиме(не самая оптимальная реализация).
